Proyectos

Sitio de Recetas

Un sistema web pensado para reemplazar las cartas físicas en bares y restaurantes. Permite a los clientes acceder al menú desde su teléfono escaneando un código QR, visualizar los platos con fotos y descripciones detalladas, filtrar por categorías (como entradas, principales, postres o bebidas) y conocer precios actualizados. Ideal para reducir costos de impresión, mejorar la experiencia del cliente y facilitar la actualización de productos y promociones en tiempo real.

Tarjeta de presentacion

Una tarjeta interactiva que reemplaza las tradicionales tarjetas impresas. Incluye nombre, profesión, foto y enlaces directos a redes sociales, correo electrónico y WhatsApp para contacto instantáneo. Puede compartirse fácilmente a través de un enlace o código QR, eliminando la necesidad de impresión y permitiendo su actualización en cualquier momento. Ideal para profesionales y emprendedores que buscan una forma moderna y ecológica de presentarse.

App de Lista de Tareas

Una herramienta web sencilla y efectiva para organizar actividades diarias. Permite agregar, marcar como completadas y eliminar tareas de forma rápida e intuitiva.
